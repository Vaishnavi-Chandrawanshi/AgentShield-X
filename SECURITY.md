# Security Policy

## Supported Versions

We actively update and patch security bugs on the current release branch.

| Version | Supported |
| :--- | :--- |
| v3.x |  Yes |
| v2.x | ❌ No |
| v1.x | ❌ No |

## Reporting a Vulnerability

If you identify a security vulnerability or exploit vector in AgentShield-X, please **do not report it via public GitHub issues**. Instead, report it privately:

1. Send an email to **vaishnavi@example.com** with details of the vulnerability.
2. Include reproduction steps, a proof of concept payload, and potential impact assessments.
3. We will acknowledge receipt of your report within 48 hours and coordinate remediation before public disclosure.

Thank you for helping keep AgentShield-X and its users secure!
